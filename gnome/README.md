# GNOME Layer

Run this after common bootstrap/Chezmoi and the GUI layer when the machine uses GNOME.

```bash
bash gnome/run.sh
```

Current scope:

- GNOME desktop portal for screen sharing
- GNOME workspace and app keybindings
- Voxtype, `ydotool`, and related Wayland input setup

The GUI layer owns desktop applications such as browsers, terminals, editors, and audio/video tools.
